# 💼 Portfólio Pessoal

Aplicação web desenvolvida para apresentar meus projetos, habilidades e informações profissionais de forma moderna, responsiva e acessível.

🔗 **Acesse o portfólio:**
https://miqueias-eduardo.github.io/portfolio/

---

## 🚀 Sobre o projeto

Este portfólio foi criado com o objetivo de centralizar minhas informações como desenvolvedor, proporcionando uma navegação intuitiva e uma apresentação clara dos meus trabalhos.

A aplicação foi desenvolvida utilizando apenas tecnologias base da web, com foco no domínio de fundamentos, organização de código e experiência do usuário.

---

## ✨ Funcionalidades

- Navegação suave entre seções (scroll behavior)
- Menu responsivo para dispositivos móveis
- Alternância de tema (claro/escuro)
- Exibição de projetos com imagens e descrições
- Formulário de contato com validação de dados
- Download de currículo diretamente pelo site
- Interface dinâmica com manipulação de DOM

---

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES Modules)**
- Manipulação de DOM
- Responsividade (Mobile-first)
